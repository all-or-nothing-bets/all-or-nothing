## Quickstart

Install packages (use Node 12)

```bash
yarn install
```

Start local Hardhat blockchain:

```bash
yarn chain
```

In another terminal window, deploy smart contracts:

```bash
cd core
yarn deploy
```

Start React app

```bash
cd interface
yarn start
```
